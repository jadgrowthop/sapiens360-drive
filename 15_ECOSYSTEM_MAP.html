Flux de Donnees

Comment les donnees d'un lead circulent a travers l'ecosysteme, etape par etape.

Lead commente "SAPIENS" sur Instagram
Un prospect voit un reel de Geoffrey et commente le mot-cle declencheur.
Instagram
ManyChat capture: prenom Instagram + interaction
Le flow ManyChat se declenche automatiquement, capture le prenom et le type d'interaction.
ManyChat
Lead donne son email → ManyChat webhook → GHL cree contact
L'email est capture, le webhook POST envoie les donnees a GHL qui cree le contact avec les tags initiaux.
ManyChat GHL manychat-qualifie
ManyChat envoie VSL → tag vsl-envoyee
Le lead recoit le lien de la VSL. Le tag est ajoute dans GHL pour le tracking.
ManyChat vsl-envoyee
Lead booke Calendly → webhook → GHL tag rdv-reserve
Le lead prend rendez-vous. Le webhook Calendly tague le contact et remplit le custom field "condition".
Calendly GHL rdv-reserve
GHL WF2 s'active (emails pre-call)
Le workflow 2 envoie la sequence d'emails de preparation avant le call de closing.
GHL Workflow
Call Jad → close / no-close / no-show → tags manuels
Jad effectue le call de closing. Le resultat est tague manuellement dans GHL.
Zoom GHL
Si close: Whop paiement → webhook → GHL tag client
Le client paye via Whop. Le webhook Netlify transmet a GHL qui ajoute le tag "client".
Whop Netlify GHL client
Geoffrey notifie → onboarding → Skool + WhatsApp + Fatome
Geoffrey lance l'onboarding: acces Skool, groupe WhatsApp, premier call Fatome planifie.
Skool WhatsApp Fatome

Troubleshooting

"Si X casse, fais Y" — guide de depannage rapide pour chaque point critique.

ManyChat ne repond plus
Verifier
  • Flow actif?
  • Trigger keywords corrects?
  • Compte Instagram connecte?
Fix
  • Reconnecter Instagram dans ManyChat settings
GHL n'envoie pas d'emails
Verifier
  • Workflow actif?
  • Tag correct?
  • Email du contact present?
  • Deliverabilite?
Fix
  • GHL > Workflows > le workflow specifique > Run History
Calendly webhook ne tague pas dans GHL
Verifier
  • Webhook URL correcte?
  • Custom fields mappes?
Fix
  • Tester le webhook manuellement via Calendly test event
Whop payment pas detecte
Verifier
  • Webhook URL Netlify active?
  • Function deploiee?
Fix
  • Netlify > Functions > whop-webhook logs
Emails arrivent en spam
Verifier
  • DKIM/SPF/DMARC configures?
  • Domaine pas blackliste?
Fix
  • Voir guide deliverabilite dans /guides/DELIVERABILITE_EMAIL.html
Form backfill ne fonctionne pas
Verifier
  • Cron com.sapiens360.form-backfill actif?
  • Python script /automation/backfill_form_to_customfields.py operationnel?
Fix
  • Relancer le cron, verifier les logs dans /automation/logs/
